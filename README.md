# glint
glint 是一款golang开发的web漏洞主动(被动)扫描器，是目前为止跟上主流技术的开源工具,如有一下功能:

1.xss AST语义检测

2.SQL 注入检测 （正在重构）

3.xray poc 脚本检测（这个偷懒主要参照 https://github.com/jweny/pocassist 
)

4.基于浏览器的爬虫主动扫描

5.被动扫描

6.csrf 检测

7.ssrf 检测 （正在重构）

8.jsonp ast语义检测

9.Xxe 实体注入检测 支持回显和反链平台 （正在重构）

10.CRLF 检测

11.CORS 跨域共享检测
